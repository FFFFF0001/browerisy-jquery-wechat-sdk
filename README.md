## 下载

``` bash
git clone https://github.com/xulayen/browerisy-jquery-wechat-sdk.git & cd browerisy-jquery-wechat-sdk
```

## 安装

``` bash
npm install
```

## 启动

``` bash
npm run start
```

``` js
http://localhost:3000/index.html
```

## 调试

``` bash
npm run watch
```


## 发布

``` bash
npm run build
```
